ORGANIZER GUIDE
Step-by-step from setup to results
Setup phase
Registration & publishing
Competition day
— Setup —
1
Create competition
Fill in name, date, city, country and Q coefficient. Created with DRAFT status — not visible to the public yet.
My Competitions → + New
2
Add divisions
Create weight/age categories: Men Open, Women Open, Masters 40+, U23, -105kg etc. Each division has its own start list and results.
Divisions tab
3
Add disciplines
Add all events: Log Lift, Atlas Stones, Deadlift, etc. Set the order number for each. The last discipline can be marked as Final — start order uses reverse standings.
Disciplines tab
4
Add judges
Search and assign judges: Head Judge, Judge, Technical Delegate, Referee. They appear in the Official Panel on the public page.
Judges tab
5
Upload banner & sponsors
Upload competition banner (recommended 1200×400px). Add sponsors with logos and links — displayed on the public page.
Bottom of cabinet page
— Registration & Publishing —
6
Register athletes
Option A: add manually via athlete search — assign BIB and bodyweight.
Option B: athlete applies via public page → appears as PENDING → Accept to add to start list.
Athletes tab / Registrations tab
7
Publish competition
Press PUBLISH — the public page goes live. The REGISTER NOW button appears for athletes. Can be unpublished at any time.
🌐 PUBLISH button in header
— Competition Day —
8
Draw
Open 🎲 DRAW on a projector. Press the button — athletes get lot numbers with animation. Discipline 1 start order = by lot number. Disciplines 2+ = automatic reverse by standings.
🎲 DRAW button in header
9
Open live screens
📺 LIVE SCREEN — add as browser source in OBS. Shows current athlete, discipline and standings.
🏋️ WARMUP SCREEN — display in warmup room showing who prepares next.
📺 LIVE / 🏋️ WARMUP buttons
10
Enter results
Referee enters results via Referee App for each athlete per discipline. Standings and next discipline start order update automatically.
Referee App
11
Protocol & certificates
Check Protocol tab for final results. Print official protocol via 🖨️ PROTOCOL. Send certificates via 📧 EMAIL ALL or print via 📜 CERTIFICATES.
Protocol tab / Header buttons
